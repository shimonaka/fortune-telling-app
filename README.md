# 🔮 運命の紡ぎ手 - 四柱推命×MBTI統合占いアプリ

伝統的な東洋占術（四柱推命・数秘術）と現代的な性格診断（MBTI）を組み合わせた革新的な人材分析システムです。

## ✨ 主な機能

### 1. 四柱推命とMBTIの統合分析
- **四柱推命**：年柱・月柱・日柱・時柱の詳細な解説
- **MBTI**：16タイプの性格診断と簡易診断機能
- **数秘術**：才能数・本質数・探究数の計算

### 2. 二種類の結果表示

#### 本人向け結果
- 運勢キャラクター
- 特性レーダーチャート
- 四柱推命の詳細解説（各柱の意味と影響）
- 開運ポイント（ラッキーカラー、方向、数字）
- 人間関係相性表（MBTIタイプ別のコミュニケーションのコツ）
- 過去・現在・未来の運勢の流れ（時系列表示）
- シチュエーション別の具体的な行動アドバイス

#### 雇用主向け結果
- 成長予測曲線（時系列）
- リスク管理マトリクス（離職リスク、パフォーマンスリスク、モチベーションリスク）
- アクションプラン（短期・中期・長期）
- トレーニングマトリクス（スキル開発領域）
- 具体的なコミュニケーション指針
- モチベーション管理のアプローチ方法
- チーム内での最適な役割と配置の提案
- ストレスサインの早期発見方法と対処法
- 長期的な育成プランの提案

### 3. 詳細な入力フォーム
- 生年月日（四柱推命計算用）
- 生まれた時間（オプション、時柱計算用）
- 性別選択
- MBTIタイプ選択または簡易診断機能
- 占いの目的選択（本人向け/雇用主向け）

## 🔮 使い方

1. ブラウザでアプリを開く
2. **基本情報**を入力
   - 生年月日（年、月、日）
   - 生まれた時間（オプション）
   - 性別
3. **MBTIタイプ**を選択
   - 既に知っている場合はタイプを選択
   - わからない場合は「MBTIがわからない」を選択して簡易診断を受ける
4. **占いの目的**を選択
   - 本人向け：運勢、特性、相性など
   - 雇用主向け：成長予測、リスク管理、育成プランなど
5. 「占いを実行する」ボタンをクリック
6. 詳細な分析結果が表示されます

## 📝 計算方法

### 才能数
- 生まれた日の数字を一桁になるまで足す
- マスターナンバー（11、22）はそのまま

### 本質数
- 生年月日のすべての数字を足し、一桁になるまで足す
- マスターナンバー（11、22、33）はそのまま

### 探究数
- 生まれた月と日の数字を足し、一桁になるまで足す
- マスターナンバー（11）はそのまま

### 干支
- 基準年：1984年（甲子年）
- 十干：(年 - 1984) % 10
- 十二支：(年 - 1984) % 12
